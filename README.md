# 🛒 Ecommerce App

## О проекте

**Ecommerce App** — web-приложение интернет-магазина, разработанное на Django с использованием архитектурного паттерна MTV.

Проект реализует полноценную серверную логику каталога товаров, пользовательской аутентификации, блога и системы ролей 
с разграничением прав доступа.

Приложение демонстрирует:

- Проектирование реляционных моделей данных

- Реализацию бизнес-логики через CBV

- Гибкую систему ролей и модерации

- Валидацию пользовательского контента

- Кэширование страниц

- Работающую систему поиска и пагинации

## Технологии

Python

Django

PostgreSQL

Django CBV (Class-Based Views)

Bootstrap

Кэширование (cache_page)

Пользовательская модель (CustomUser)

Ролевая модель доступа

Валидация форм и антиспам-проверки

## Основной функционал

### Каталог товаров

- Отображение списка товаров с пагинацией

- Фильтрация по категориям

- Поиск по товарам

- Страница детального просмотра товара

- Кэширование списка и карточек товаров

- Полный CRUD для каталога

- Товары имеют статусы:

    - moderation

    - published

    - archived

- Удаление реализовано через мягкую архивацию (soft delete).

### Пользовательская система

- Регистрация через email (email используется как USERNAME_FIELD)

- Авторизация

- Загрузка аватара

- Дополнительные поля (страна, телефон, подписка на рассылку)

- Отправка приветственного email при регистрации

- Реализована кастомная модель пользователя CustomUser.

### Система ролей и прав доступа

- Добавлены кастомные permissions:

    - Публикация товаров
    
    - Снятие с публикации
    
    - Публикация статей
    
    - Модерация контента

- Роли:

    - Владелец товара
    
    - Модератор продуктов
    
    - Контент-менеджер

- Контроль доступа реализован через:

    - Проверки прав во View
    
    - Переопределение dispatch()
    
    - Кастомные проверки доступа к объектам

### Валидация и бизнес-правила

- Проверка текста на спам и запрещённые слова

- Проверка корректности цены (цена > 0)

- Проверка формата и размера изображений

- Автоматическая смена статуса на moderation при редактировании

### Контакты и обратная связь

- Страница контактов с хранением контактной информации

- Форма обратной связи

- Сохранение сообщений в базе

- Flash-сообщения при успешной отправке

### Блог

- Публикация и редактирование статей

- Модерация публикаций

- Подсчёт количества просмотров

- Подсчёт количества авторов и статей

- Поиск по статьям

- Архивация вместо удаления

### Архитектурные особенности

- Использование абстрактных базовых View

- Разделение бизнес-логики и представлений

- Мягкое удаление (архивация)

- Переопределение форм в зависимости от роли пользователя

- Контекстные процессоры

- Кэширование страниц (15 минут)

- Модульная структура проекта

## Установка:

1. Клонируйте репозиторий проекта:
````
git clone https://github.com/simbarilion/Ecommerce_app.git

cd Ecommerce_app
````
2. Установите зависимости внутри каталога проекта для создания виртуального окружения и установки зависимостей:
````
poetry install
````
3. Примените миграции
```
python manage.py migrate
```
4. Запустите проект налокальном сервере
```
python manage.py runserver
```
