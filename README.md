# Проект "Ecommerce_app"

## Описание:

"Ecommerce_app" является учебным проектом по Python-разработке.
Проект содержит код для разработки простого web-приложения интернет-магазина.

Проект разрабатывается с использованием фреймворка Django и основан на паттерне MTV.
В проекте содержатся HTML-документы, являющиеся прототипами web-страниц, с подключенными к ним Bootstrap-стилями.

## Установка:

1. Клонируйте репозиторий проекта:
````
git clone https://github.com/nadezhdapopova-spec/Ecommerce_app.git
````
2. Установите зависимости внутри каталога проекта для создания виртуального окружения и установки зависимостей:
````
poetry install
````

## Использование:

Созданы базовые шаблоны для web-страниц приложения, включающие навигационное меню, футер, паггинацию.

Реализовано отображение домашней страницы:

![Home.png](data/Home.png)

Реализовано взаимодействие с PostgreSQL. На главной странице отображаются карточки товаров, добавлена паггинация.
Добавлена возможность перехода из карточки на страницу товара. Товары разделены на категории, которые отображаются в главном меню сайта.

Для каждого товара созданы отдельные страницы с подробным описанием:

![Product.png](data/Product.png)

Реализована система аутентификации пользователя, добавлена возможность регистрации пользователя и входа в систему.
Для авторизованных пользователей добавлена возможность создавать и опубликовывать карточки товаров с помощью форм:

![Product_form.png](data/Product_form.png)

Каждая карточка перед публикацией проходит валидацию на спам и запрещенные слова, а также на корректность установленной цены и загруженных изображений.

Реализовано отображение страницы с контактной информацией.
Страница "Контакты" содержит форму обратной связи, информация от ползователя через POST запрос сохраняется в базу данных:

![Contacts.png](data/Contacts.png)

В результате обработки данных формы в контроллере отображается всплывающее окно с сообщением об успешной отправке данных.

Создан раздел Блог. Для авторизованных пользователей реализована возможность не только читать, 
но и публиковать статьи, а также редактировать и отправлять в архив свои публикации.

![Blog.png](data/Blog.png)

![Blogposts_editor.png](data/Blogposts_editor.png)

Создан счетчик просмотра статьи.

![Blogpost_editor.png](data/Blogpost_editor.png)

Реализован счетчик авторов и публикаций.

На сайте работает поиск товаров и статей.

Контроллеры написаны с помощью CBV, для раздела Блог реализован полный CRUD.
